# Frontend

This directory contains the SvelteKit frontend for the Wails desktop app.

## Role

- Renders the desktop UI in SPA mode (`ssr = false`)
- Talks to the Go backend through the Wails bridge in [`src/lib/wails`](./src/lib/wails)
- Uses generated Wails bindings under `src/lib/wails/wailsjs/` and thin handwritten wrappers for app-specific behavior

## Development

```sh
pnpm install
pnpm dev
pnpm check
pnpm test:run
```

Use `pnpm dev` for browser-only UI work. The Wails wrapper expects the desktop runtime and now fails fast when backend bindings are unavailable instead of returning placeholder data.

## Structure

- `src/routes/` route entry points for the tabbed UI
- `src/lib/components/` shared UI and feature components
- `src/lib/contexts/` shared reactive contexts
- `src/lib/schema/` Zod validation for form payloads
- `src/lib/wails/` frontend contract layer for the Go backend

## Generated Files

- `src/lib/wails/wailsjs/` is generated by Wails
- `.svelte-kit/` and `build/` are local build artifacts and should not be treated as source
